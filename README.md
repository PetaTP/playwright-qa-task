# Playwright UI Automation Project

This project demonstrates basic UI automation testing using Playwright.

## What this project tests
The automated tests simulate common user interactions and verify that key functionality works as expected.

Test coverage includes:
- Navigation flows
- User interactions
- Edge case scenarios

## Tools Used
- Playwright
- JavaScript

## Purpose
This project was created as part of my transition into Quality Assurance to practice writing automated UI tests and documenting test coverage.
